02_Emnesite

Projektbeskrivelse

I opgave 02_Emnesite var målet at skabe vores eget design med et selvvalgt emne. Vi lærte, hvordan man starter et designprojekt og implementerer det i HTML, CSS og JavaScript. Det var i dette emne, vi for alvor blev introduceret til JavaScript. Derudover brugte vi Figma til at lave styletiles, moodboards og vores eget layoutdiagram.



Rolle

Projekt: Emnesite

Rolle: Programmør (primært HTML, CSS og en smule JavaScript), designer og brug af Figma

Varighed: 4 uger

Det her lærte vi

I Tema 02_Emnesite lærte vi om forskellige layoutprincipper, herunder vigtigheden af at skabe vores eget design med styletiles, finde de rette skrifttyper og farver samt oprette moodboards. Vi udførte også tests ved hjælp af Figma, hvor test personerne skulle teste vores prototype af vores hjemmeside. Derudover arbejdede vi med layoutdiagrammer, HTML og CSS.

Endelige Redesign

Det var vigtigt at anvende den viden, vi havde opnået i Tema 01, så vi kunne bruge de samme grid-principper og gøre designet så personligt som muligt. Vi havde frihed til at vælge design, herunder farver og skrifttyper. Mit emne var at lave en guide for burgerentusiaster, hvor man kunne se, hvilke burgerrestauranter jeg anbefalede samt finde links til deres hjemmesider. Jeg fokuserede meget på farve- og fontvalg, da det spiller en stor rolle i mit redesign og gør det så brugervenligt som muligt. Det endelige redesign og hvordan jeg placerede billederne i grids kan ses via linket nederst på siden.
